## Prerequisites

### Working with virtual environment

#### Create virtual environment

```bash
python -m venv venv
```

#### Activate (Unix/macOS)

```bash
source venv/bin/activate
```

#### Activate (Windows)

```bash
venv\Scripts\activate
```

#### Verify Python version

```bash
python --version
```

#### Deactivate

```bash
deactivate
```
